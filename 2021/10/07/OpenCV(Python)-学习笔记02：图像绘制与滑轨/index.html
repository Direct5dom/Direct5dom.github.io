绘图功能

OpenCV提供了一些绘图函数，如：cv.line()、cv.circle()、cv.rectangle()、cv.ellipse()、**cv.putText()**等。

在上面这些函数，能看到一些相同的参数：

  • img：你想画的图片

  • color：形状的颜色，如 BGR，它是一个元组，例如：蓝色(255,0,0)。对于灰度图，只需传一个标量值。

  • thickness： 线或圆等的厚度。如果传 -1 就是像圆的闭合图形，它将填充形状。_ 默认 thickness = 1_

  • lineType：线条类型：

    Enumerator 说明
    FILLED Python: cv.FILLED
    LINE_4 Python: cv.LINE_4 4-connected line
    LINE_8 Python: cv.LINE_8 8-connected line
    LINE_AA Python: cv.LINE_AA antialiased line

参数 说明
img 图像
center 椭圆的中心
axes 椭圆主轴大小的一半

endAngle 椭圆弧的结束角度（以度为单位）

Enumerator 说明

LINE_AA Python: cv.LINE_AA antialiased line

    FONT_HERSHEY_SCRIPT_COMPLEX Python: cv.FONT_HERSHEY_SCRIPT_COMPLEX more complex variant of FONT_HERSHEY_SCRIPT_SIMPLEX

这里我们简单的创建一个程序，会显示一个窗口，其中有三个滑轨，用来决定RGB色彩。该色彩与

这里，我们将创建简单的程序，显示你指定颜色。 你有一个显示颜色的窗口和四个轨迹栏，用来指定 B，G，R 颜色和是否显示颜色。

官方文档这里解释了一下函数定义：

For cv.createTrackbar() function, first argument is the trackbar name, second one is the window name to which it is attached, third argument is the default value, fourth one is the maximum value and fifth one is the callback function which is executed every time trackbar value changes. The callback function always has a default argument which is the trackbar position. In our case, function does nothing, so we simply pass.

对于cv.createTrackbar()函数，第一个参数是轨迹栏名称，第二个参数是它所附加的窗口名称，第三个参数是默认值，第四个参数是最大值，第五个参数是执行的回调函数 每次轨迹栏值更改时。 回调函数总是有一个默认参数，即轨迹栏位置。 在我们的例子中，函数什么都不做，所以我们只是通过。

滑轨除了实现它本来的作用，还可以当作按钮或者开关。OpenCV在默认情况下是没有按钮功能的。因此只能用滑轨做一些这样的功能。
